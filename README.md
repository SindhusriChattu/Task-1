# Task 1: Data Cleaning and Preprocessing

## Dataset

Sales Export 2019–2020 Dataset

## Objective

Clean and prepare the raw dataset by identifying and handling data quality issues such as missing values, duplicate records, inconsistent formatting, and incorrect data types.

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Steps Performed

1. Loaded the Sales Export 2019–2020 dataset.
2. Explored the dataset using `head()`, `tail()`, and `info()`.
3. Checked for missing values using `isnull()`.
4. Checked for duplicate records and removed them if present.
5. Standardized text values by removing extra spaces and applying consistent formatting.
6. Converted date columns to a consistent date format.
7. Renamed column headers to lowercase and replaced spaces with underscores.
8. Checked and validated data types for all columns.
9. Saved the cleaned dataset as `sales_cleaned.csv`.

## Files Included

* `Task 1 - Data Cleaning and Preprocessing.ipynb`
* `Sales-Export_2019-2020.csv`
* `sales_cleaned.csv`
* `README.md`

## Outcome

Successfully cleaned and prepared the dataset for further analysis and visualization. The final dataset is consistent, structured, and ready for data analysis tasks.
